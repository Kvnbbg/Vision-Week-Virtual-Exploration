# Vision Week – Offre B2B VR “incomparable à Fiverr”

## 1) Diagnostic marché & angle gagnant

**Pourquoi ce n’est pas Fiverr (5 arguments business)**
1. **Spécialisation verticale VR loisirs** : workflows réservations, créneaux, capacité, licences, parc matériel, staffing VR = expertise métier non générique.
2. **Exploitation & récurrence** : focus sur le chiffre d’affaires récurrent (bookings, no-shows, up-sell), pas un simple site livré puis abandonné.
3. **Maintenance opérationnelle** : monitoring, mises à jour sécurité, conformité paiement (PSP), backups, SLA—ce qu’un freelance “one-shot” ne maintient pas.
4. **Conformité & risques** : gestion RGPD (consentement, export), traçabilité paiements, logs opérationnels ; réduction du risque légal.
5. **ROI mesurable** : KPIs d’occupation, panier moyen, coût d’acquisition, conversion web → réservation ; optimisation continue.

**Risques majeurs & parades**
- **Risque : adoption lente par le staff** → *Parade* : onboarding structuré + workflows simplifiés + “mode staff” ultra rapide.
- **Risque : dépendance à un PSP** → *Parade* : architecture PSP-agnostique (Stripe, Adyen, Alma) + fallback facturation.
- **Risque : intégration difficile avec outils existants** → *Parade* : connecteurs CSV/API + migration assistée + tests de reprise.
- **Risque : saisonnalité forte** → *Parade* : offres promo/abonnements, campagnes email ciblées, analytics par période.
- **Risque : concurrence low-cost** → *Parade* : positionnement sur ROI + SLA + expertise verticale + time-to-value.

## 2) Positionnement (incomparable)

- **Tagline** : *La plateforme d’exploitation qui remplit vos créneaux VR, pas seulement votre site.*
- **Promesse principale** : *Augmenter votre taux d’occupation et votre CA par session grâce à une plateforme verticale VR, pilotée par des données.*
- **3 preuves (mécaniques)** :
  1. **Optimisation des créneaux** : moteur de capacité + upsell automatique (pack boissons, anniversaire, team-building).
  2. **Flux réservé → payé** : paiement en ligne + rappel automatique = réduction des no-shows.
  3. **Tableaux de bord exploitables** : KPI d’occupation, panier moyen, conversion par canal.
- **Anti-promesse** : *Nous ne faisons pas de sites vitrines “sur-mesure créatif” sans objectifs business mesurables.*

## 3) ICP + décideurs + pains

### ICP 1 — Salle VR indépendante (1 site)
- **Décideur** : fondateur / gérant.
- **Douleurs** : créneaux creux, staff débordé, gestion manuelle des réservations, no-shows.
- **Métriques** : taux d’occupation, CA/heure, no-show %, avis clients.
- **Objections** : “Trop cher”, “on n’a pas le temps de changer”, “on a déjà un outil”.

### ICP 2 — Chaîne multi-sites (2–10 sites)
- **Décideur** : directeur des opérations + DG.
- **Douleurs** : incohérence des process, reporting fragmenté, marge sous pression, pilotage multi-sites.
- **Métriques** : marge par site, taux de conversion web, NPS, coût d’exploitation.
- **Objections** : “Intégration lourde”, “besoin de centralisation”, “sécurité & RGPD”.

### ICP 3 — Centre de loisirs multiservices (VR + escape + arcade)
- **Décideur** : responsable expérience / responsable digital.
- **Douleurs** : cross-sell faible, réservation multi-activités complexe, cash desk peu fiable.
- **Métriques** : panier moyen global, taux de rebooking, temps staff/booking.
- **Objections** : “Pas sûr que la VR justifie une plateforme dédiée”.

## 4) Offre packagée en 3 niveaux (Starter / Pro / Premium)

### Starter
- **Pour qui** : salle VR indépendante.
- **Livrables** : réservation en ligne, paiement, catalogue d’expériences, admin staff basique, emails transactionnels.
- **Résultat attendu** : +15–25% de réservations en ligne sous 60 jours.
- **Support / SLA** : email 48h ouvrées.
- **Prix conseillé (France)** : setup 2 000–4 000€ + abonnement 290–590€/mois.

### Pro
- **Pour qui** : salles VR en croissance ou 2–3 sites.
- **Livrables** : tout Starter + analytics avancés, CRM léger, automatisations email, segmentation clients, reporting mensuel.
- **Résultat attendu** : +20–35% taux d’occupation, -30% no-shows en 90 jours.
- **Support / SLA** : email 24h + chat.
- **Prix conseillé** : setup 5 000–9 000€ + abonnement 790–1 490€/mois.

### Premium
- **Pour qui** : chaînes multi-sites / centres loisirs.
- **Livrables** : tout Pro + multi-sites, rôles avancés, API/BI, intégrations, SLA dédié.
- **Résultat attendu** : +25–40% CA par site, pilotage unifié.
- **Support / SLA** : support prioritaire + CSM dédié.
- **Prix conseillé** : setup 12 000–25 000€ + abonnement 2 000–5 000€/mois.

**Add-ons (à la carte)**
- Campagnes acquisition (ads + landing)
- Modules fidélité/abonnements
- Kiosque self-service / QR check-in
- Formation staff sur site

**Conditions**
- Setup non remboursable, engagement 12 mois.
- Résiliation : préavis 60 jours.
- Migration data incluse jusqu’à X clients.
